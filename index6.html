Sites

  Sites podem ser acessados a partir de navegadores, navegadores podem ser usados por dispositivos moveis ou outros dispositivos como computadores e outros.

Internet

  A internet e necessaria para acessar qualquer site, alguns jogos precisam de internet assim como os sites, não são todos mas tem alguns, caso tente entrar em um site desconectado no caso sem internet vai dar uma tela de erro pedindo para ver se você está conectado com a internet.

HTML

  Sites são feitos com que linguagem de programação? Nenhuma, sites são feitos com HTML uma linguagem de marcação, as unicas linguagens que podem ser usadas na criação de sites é o javascript e o PHP claro você não pode criar um site so com um arquivo javascript mas, é possivel criar um site só com um arquivo PHP.

Css

  Css é a forma de estilizar seu site, você pode criar um arquivo.css ou criar no codigo HTML uma tag style, o metodo mais recente é criar um arquivo.css a tag style é meio desatualizada.
